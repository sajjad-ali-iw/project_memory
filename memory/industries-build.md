**Staging measurement tip:** staging has an aggressive scroll-reset/loader that prevents scrolling+screenshots. Curl the page to the local webroot (`curl ... -o _mirror.html` with Bash `dangerouslyDisableSandbox:true` since the sandbox blocks DNS), strip `<script>` tags with perl, open `http://indexworld-revamp.local/_mirror.html`. Scroll is still locked (body overflow) — to view a section, JS-move it to `document.body` and `display:none` the siblings, then screenshot. getComputedStyle/getBoundingClientRect on the static mirror are reliable. Delete the mirror file when done.
